# Changelog

All notable changes to Green Basket POS will be documented here.

This project follows a simple changelog style that is easy for beginners to
understand.

## [1.0.0] - 2026-07-14

### Added

- Offline Electron POS application
- Sale, Inventory, Reports, and Settings pages
- Product image upload
- Cart checkout flow
- Held carts
- Receipt preview and print support
- Inventory stock adjustments
- Sales reporting
- Backup and restore
- Installer packaging with electron-builder
- Beginner-friendly open-source documentation

### Notes

- This is the first open-source-ready version of the project.
