# Support

Need help with Green Basket POS?

## For Users

Open a GitHub issue if:

- The app does not start
- A feature is not working
- You found a layout problem
- You need help building the app

Please include:

- Your operating system
- Node.js version
- npm version
- Steps you followed
- Screenshots if the issue is visual

## For Beginner Contributors

It is okay if you are new. You can ask questions about:

- Setting up the project
- Understanding a file
- Choosing a first issue
- Running checks
- Opening a pull request

When asking for help, share what you tried and what error you saw. That makes it
much easier for maintainers to guide you.
