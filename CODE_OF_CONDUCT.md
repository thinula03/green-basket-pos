# Code of Conduct

## Our Pledge

We want Green Basket POS to be a welcoming project for beginners and experienced
developers. Everyone should be able to learn, ask questions, and contribute
without being treated unfairly or disrespectfully.

## Expected Behavior

- Be kind and patient.
- Explain ideas clearly.
- Give constructive feedback.
- Respect different experience levels.
- Help beginners learn instead of making them feel small.
- Assume good intent when possible.

## Unacceptable Behavior

- Harassment, insults, or personal attacks
- Discrimination of any kind
- Sexual language or unwanted attention
- Trolling or intentionally disruptive comments
- Sharing private information without permission

## Enforcement

Project maintainers may remove comments, close issues, reject pull requests, or
block users who break this code of conduct.

If you see behavior that makes the project unsafe or unwelcoming, contact the
maintainers through the repository's issue or discussion channels.
