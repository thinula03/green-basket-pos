# Roadmap

This roadmap lists ideas for future contributors. It is not a strict promise;
it is a guide for what would make the app more useful and educational.

## Beginner Friendly

- Add screenshots to the README
- Add more sample grocery products
- Improve empty states on each page
- Improve keyboard shortcut documentation
- Add comments to complex renderer logic
- Add simple UI polish issues labeled `good first issue`

## Product Improvements

- Product import and export with CSV
- Better receipt customization
- Barcode label printing
- Customer management
- Supplier management
- Purchase orders
- Stock reorder alerts
- Daily closing report
- Multi-currency formatting improvements

## Technical Improvements

- Add automated tests for pricing and cart calculations
- Split large renderer logic into smaller modules
- Add linting and formatting tools
- Add GitHub Actions for CI
- Add sample screenshots to pull request checks
- Improve packaging documentation for each operating system

## Documentation Improvements

- Add a beginner Git and GitHub tutorial
- Add a guide for creating a first pull request
- Add screenshots for setup steps
- Add a guide for customizing the app for a real shop
